# java-cli-gradle-spring-jbehave-mockito-testng-car-data

## Description
A POC for a spring app using testng,
jbehave and mockito framework.

## Tech stack
- java
- gradle
	- spring
  - testng
  - jbehave
	- mockito

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
